PyQt5快速入门(一)

前言

为什么选择PyQt5作为GUI框架?

  • API与Qt一致, 学会PyQt后再使用qt很简单
  • 开发迅速, 可视化操作,使用designer快速拖拽布局进行调试
  • 可以将文件打包成exe进行发布

本节课内容（假设已经掌握python语法）

  • 搭建PyQt5环境
  • 测试PyQt5环境

本节课使用环境

  • python 3.6.6
  • IDLE(一般安装python后自带)
